## 1238. 循环码排列

### 题目传送门

[点击这里](https://leetcode.cn/problems/circular-permutation-in-binary-representation/)

### 解题思路

这道题的题意就是格雷码，其法则是保留二进制码的最高位作为格雷码的最高位，而次高位格雷码为二进制码的高位与次高位相异或，而格雷码其余各位与次高位的求法相类似。

### 代码

```go
func circularPermutation(n int, start int) []int {
	g := make([]int, 1<<n)
	j := 0
	for i := range g {
		g[i] = i ^ (i >> 1)
		if g[i] == start {
			j = i
		}
	}
	return append(g[j:], g[:j]...)
}

```
